3. create Model(our table class that talk to database) in ORM 

3.1 crate model (class that working with database) 

Create model link : https://laravel.com/docs/11.x/eloquent

>>php artisan make:model Product

it will create model Product in the App -> Models -> Product.php 

==> 

<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class Product extends Model
{
    //
}


In case our Model name different from our table we put : protected 

==> 

<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class Product extends Model
{
    
    // protected $table = "products";

protected $fillable = ['name', 'price', 'description']; // to make our table mapping to the model

}

3.4 in the Web.php we want to crate with ORM 

use App\Models\Product; // import our Model Produts 

Route::get('/products',function(){
    $products = Product::all();
    return response()->json($products);
});
