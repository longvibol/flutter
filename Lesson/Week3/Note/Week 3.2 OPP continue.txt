- Inheritance :ការផ្ទេរមរតក គឺជាការ share នៅ datamember or sharing Method and property to other class (super class or child):
Parent : superclass or base class
Child : subclass or child class 

Ex: we have class Employee 

class Employee {
  late double id;
  late String fullName;
  late String gender;
  late double phone;

  void register(
      {required double id,
      required String fullName,
      required String gender,
      required double phone}) {
    this.id = id;
    this.fullName = fullName;
    this.gender = gender;
    this.phone = phone;
  }
}

--------------------------------------------  

ចឹងដើម្បី Inheritance យើងប្រើយ Keyword : extends 
Ex:

class User extends Employee {
  late String userName;
  late String passWord;

  void registerUser(
      {required double id,
      required String fullName,
      required String gender,
      required double phone,
      required String userName,
      required String passWord}) {
    this.userName = userName;
    this.passWord = passWord;
  }
}

We can see that it is shorter than superclass : in User class we just add username and passpord :
