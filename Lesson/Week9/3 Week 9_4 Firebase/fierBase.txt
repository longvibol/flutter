Deploy Website to Firebase : 

1. install firebase tool with node :

>> npm install -g firebase-tools

2. go to our app want to deploy : 

login to the firebase website with oun gmail account 

Link: 

https://console.firebase.google.com/u/0/?utm_source=email&utm_medium=newsletter&utm_campaign=firebase-welcome

==> Create a project name : fackstore-vibol 

==> the analytic by google 

3. in our computer login with firebase 

>> firebase login

after successful we create firebase json in out app 

4. Build firebase in 

>> firebase init ( create firebase directory folder)

*** last Step ***

in the firebase.json change the hosting:

"hosting": {
    "public": "public",
    "ignore": [
      "firebase.json",
      "**/.*",
      "**/node_modules/**"
    ],

==> To 

   "public": "build/web",

-------------------------------------

Last command : 

>> flutter build web 

>> firebase deploy
