គេប្រើញឹកញាប់គឺ 

- mainAxisAlignment គឺយើងរៀបជួរដេកដើម្បីទុក space រវាងគ្នាវា​
Ex: mainAxisAlignment: MainAxisAlignment.spaceEvenly,


- crossAxisAlignment រៀបវាជាជួរឈរវិញ

Ex:

body: Column(
        children: [
          Container(
            color: Colors.red,
            height: 300,
            child: Row(
              // crossAxisAlignment: CrossAxisAlignment.end,
              // mainAxisAlignment: MainAxisAlignment.center,
              crossAxisAlignment: CrossAxisAlignment.center,
              mainAxisAlignment: MainAxisAlignment.center,
              children: [
                Container(
                  height: 200,
                  width: 200,
                  color: Colors.green,
                )
              ],
            ),
          )
        ],

      ), 

--Example : CrossAxisAlignment លើក្រោម and MainAxisAlignment ឆ្វេងស្តាំ
